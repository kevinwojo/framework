Request

Extends \Symfony\Component\HttpFoundation\Request

Request handler replaces the default PHP global variables and functions by an object-oriented layer.

A number of methods are inspired by or follow Joomla syntax to maintain compatibility.

Further inspiration for Laravel's Request class http://laravel.com
