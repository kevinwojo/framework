SessionToken

Implements \OAuth2\GrantType\GrantTypeInterface, \OAuth2\ClientAssertionType\ClientAssertionTypeInterface

Session token grant type class

By implementing "ClientAssertionTypeInterface" client id/secret enforcement and validation is moved to this class itself and should be done in the "validateRequest" method. For simplicity sake when requesting a token via session (through ajax) no client id/secret is required.
