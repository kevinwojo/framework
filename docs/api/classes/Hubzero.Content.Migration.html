migrate(string $direction = 'up', boolean $force = false, boolean $dryrun = false, boolean $listAll = false, boolean $logOnly = false) : boolean

Arguments

$direction

string

Direction to migrate (up or down)

$force

boolean

Run the update, even if the database says it's already been run

$dryrun

boolean

Run the udpate, but only display what would be changed, wihthout actually doing anything

$listAll

boolean

List all files found, not just those needing to be run

$logOnly

boolean

Run the update, and mark as run, but don't actually run sql (usefully to mark changes that had already been made manually)

Response

boolean
