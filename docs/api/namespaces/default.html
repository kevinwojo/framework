AdminController¶

Base admin controller for components to extend.

Accepts an array of configuration values to the constructor. If no config passed, it will automatically determine the component and controller names. Internally, sets the $database, $user, $view, and component $config.

Executable tasks are determined by method name. All public methods that end in "Task" (e.g., displayTask, editTask) are callable by the end user.

View name defaults to controller name with layout defaulting to task name. So, a $controller of "One" and a $task of "two" will map to:

/{component name} /views /one /tmpl /two.php

Identicon is a library which generate an identicon image based on a string.

  • Images are generated in PNG format with transparent background.
  • The string can be an email, an IP address, a username, an ID or something else.

Usage:

$identicon = new Identicon();

// Generate image and display $identicon->displayImage('foo');

// Get image data $imageData = $identicon->getImageData('bar');

// Generate and get the base 64 image uri ready for integrate into an HTML img tag. $imageDataUri = $identicon->getImageDataUri('bar');
